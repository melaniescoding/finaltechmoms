Resources:

2. Applying for Social Security Benefits

Apply at age 18+ or earlier if disabled:

  • Gather documents:
    • Birth certificate Retrieve Birth
    • Medical records
    • Guardianship papers
  • Apply online: Apply for Benefits
  • Call Social Security: 1-800-772-1213

3. Applying for Disability Benefits

Follow these steps:

  • Complete application: Disability Application
  • Submit supporting medical documentation
  • Follow-up on application status
  • Appeals process if denied: Appeals Process
  • Apply For Medicaid Apply For Medicaid
